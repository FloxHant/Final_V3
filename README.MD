Интерактивный телеграм-бота для онлайн-школы!
Бот имеет  интерактивные элементы - кнопки, для удобного взаимодействия со студентами.
Студенты имеют возможность запросить расписание всех уроков школы, нажав на соответствующую кнопку.
Бот может предоставить простой и интуитивный понятный интерфейс для студентов.
