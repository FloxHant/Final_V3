Интерактивного телеграм-бота для онлайн-школы!
Бот поддерживает интерактивные элементы - кнопки, для удобного взаимодействия со студентами.
Студенты имеют возможность запросить расписание всех уроков школы, нажав на соответствующую кнопку.
Бот  предоставляет простой и интуитивно понятный интерфейс для студентов.
